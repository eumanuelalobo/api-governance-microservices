# Governança de APIs em Microsserviços

Este repositório contém uma análise técnica sobre a estruturação e segurança de sistemas distribuídos.

### 🔍 Destaques do Estudo:
- **Padrões de Comunicação:** Comparativo entre arquiteturas REST e GraphQL.
- **Centralização:** Uso de portais de desenvolvedor (como Backstage.io) para documentação unificada.
- **Segurança e Contratos:** Implementação de testes de contrato com a ferramenta **Pact** para garantir a integridade das interfaces.
- **Documentação:** Aplicação do padrão **OpenAPI** para escalabilidade e manutenção.
